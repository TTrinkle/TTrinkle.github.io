About Loops

Loops are a tool to make the computer do one task repetitively until a condition is met.
The main types are:

For Loops
While
Do (specific to ruby)
Each (specific to Ruby)

In Ruby, loops are contained with the loop definition and the keyword 'end'.

Javascript uses enclosures like brackets {} and parentheses () to keep track of what is inside of the loop.
To the left and right are some comparisons on how to define and run loops in each language.
