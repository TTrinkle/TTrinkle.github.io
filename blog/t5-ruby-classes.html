Ruby Classes

March 5th, 2015

This week we focus heavily on using Classes within Ruby. Classes have a state and a behavior, and will always create an object.

In order to create a new instance of the class you will call the class name followed by .new. If you plan to pass parameters to the class you must initialize the class with those expected parameters. Initialize will only execute when you use the .new method.Using the .new method will create a new object of that class. You can then call one of the method’s you’ve defined on that object to see the result.

class NewStudent def initialize(name, age, math_score) @name = name @age = age @math_score = math_score end def student return @name end def grade if @age >12 @grade = "5th" elsif @age > 10 @grade = "3rd" else @grade = "1st" end return "#{@name} is in the #{@grade} grade." end def math_class if @math_score > 80 mathclass = "honors" else mathclass = "regular" end return "#{@name} is in the #{mathclass} math class." end end Tommy = NewStudent.new("Tommy", 9, 70) Jill = NewStudent.new("Jill", 13, 90) Bobby = NewStudent.new("Bobby", 11, 88) p Tommy.math_class p Jill.grade p Bobby.grade p Bobby.math_class
